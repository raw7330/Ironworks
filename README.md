## アプリ名

Ironworks

## サイト概要

鉄工所のマッチングサービス

### サイトテーマ

鉄工所業務の最適化

### テーマを選んだ理由

前職で鉄工所への営業をしていた際、こんなサービスがあったらいいなという思いからこのテーマを選定致しました。
社内でできないことは、外注でしないといけない。今まで取り扱ったことのない技術を要する場合、新たに外注を探さないといけない。
その手間を省くことで、業務の簡素化ができ、他の業務に専念できると考えたため。

### ターゲットユーザ

鉄工所、メーカー

鉄工所（機械加工を取り扱う）　一次下請けメイン
メーカー　工作機械や建設機械、社内設備など機械工部品を必要としているメーカー

### 主な利用シーン

社内の技術や保有設備で対応できない場合の外注探し
